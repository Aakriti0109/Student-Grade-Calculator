# 📘 Student Grade Calculator (Java)

## 📌 Overview

The **Student Grade Calculator** is a Java-based application designed to manage student records and evaluate academic performance. It demonstrates fundamental concepts of **Object-Oriented Programming (OOP)** such as classes, objects, methods, and arrays.

This project allows users to input student details, record grades, calculate averages, and determine pass/fail status.

---

## 🚀 Features

* Add multiple students with name and student ID
* Store and manage student grades
* Calculate individual **average grades**
* Determine **pass/fail status**
* Display student grade reports
* Compute **class average**

---

## 🛠️ Technologies Used

* Java (Core Java)
* OOP Concepts
* Arrays & Methods
* Console-based Input/Output

## 🧑‍💻 How It Works

1. User enters student details (name, ID)
2. Inputs grades for each student
3. Program calculates:

   * Average grade
   * Pass/Fail status
4. Displays results for each student
5. Calculates overall class average

---

## ▶️ How to Run

### Using Command Line

```bash
javac Student.java Main.java
java Main
```

### Using IntelliJ IDEA

1. Open the project
2. Navigate to `Main.java`
3. Click **Run ▶️**
